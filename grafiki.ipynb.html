steam
#%% 
plt.figure(figsize=(25,5))
plt.plot(steam['game'].head(10),steam['peak_players'].head(10),label = 'Пиковое число игроков',color = 'yellow', marker = 'o')
plt.title('Статистика пикового значения игроков')
plt.xlabel('Название игры')
plt.ylabel('Число пикового значения игроков')
plt.grid()
plt.legend()
plt.show()

Линейный график статистики пикового значения игроков. Было взято 10 игр из датасета. Можно сделать вывод, что из всех 10 игр наибольшее количество игроков было в Resident Evil 4. 
График строился с помощью библиотеки matplotlib.pyplot. steam['game'].head(10) и steam['peak_players'].head(10) берёт первые 10 значений из начала датасета. Label задает название 
величины для легенды. Color это цвет графика, marker это точки на графике. plt.xlabel и plt.ylabel это названия осей x и y. plt.legend показывает легенду графика. plt.show показывает сам график. plt.title показывает название графика. plt.figure используется для создания фигуры, plt.grid используется для показа сетки.
